Version 0.3

~August 4, 2025 ¬∑ Initial Prototype

Initial Release

First public prototype release. Developers note it was made by Free Lives team in "about 3 weeks" as a rapid prototype with intentionally rough edges to get player feedback quickly.

üéÆ Core Features

  • Basic dress design system (bodice, sleeves, skirt selection)
  • Fabric cutting mechanics with pattern placement
  • Sewing mini-game with quality scoring
  • Customer request system
  • Limited fabric and accessory options
  • 5 commissions to complete in prototype

‚úÖ Early Bug Fixes

  • Fixed confetti audio (was "violent/jarring")
  • Fixed pattern naming confusion (skirt pieces had identical names)
  • Multiple scissor and cutting bugs fixed

üéØ Development Philosophy

Developers: "As happens with prototypes, we work in a very quick, dirty manner just to have something working that we can put in front of people asap... and a lot of things tend to be buggy." This rapid iteration approach continued through all versions.
