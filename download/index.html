2. Mac Installation Fix (Gatekeeper Warning)

If you're on macOS, you'll encounter a security prompt when trying to open Dressmaker. Here's why and how to fix it:

Why This Happens

Dressmaker is not notarized by Apple (the developers don't have an Apple Developer account). macOS's Gatekeeper security feature blocks unnotarized apps by default to protect users from malware.

Why use the official page? The itch.io page is the official source from elyaradine, sarandipityx, and raithza. Avoid reuploaded builds and APK mirrors.

The Error Message You'll See

"Dressmaker.app can't be opened because Apple cannot check it for malicious software."

✅ Solution: Bypass Gatekeeper (Official Method)

Follow these steps from Apple Support:

  1. Try to open Dressmaker.app (you'll see the error)
  2. Click "OK" to dismiss the error
  3. Go to System Settings (or System Preferences on older macOS)
  4. Click Privacy & Security
  5. Scroll down to the Security section
  6. You'll see a message: "Dressmaker.app was blocked from use because it is not from an identified developer"
  7. Click "Open Anyway"
  8. A confirmation dialog will appear—click "Open" again
  9. Enter your Mac password if prompted
  10. The game will now launch!

💡 Alternative Method (Control + Click)

Some users prefer this faster method:

  1. Hold the Control key and click on Dressmaker.app
  2. Select "Open" from the context menu
  3. In the dialog that appears, click "Open" again

Source: Apple Support - Safely open apps on your Mac

📝 Note for Future Versions

Once you've allowed Dressmaker to run, macOS will remember this. You won't need to repeat these steps for future launches or updates.

4. Common Installation Issues

❌ "Windows protected your PC" (SmartScreen)

Cause: Windows Defender SmartScreen blocks unrecognized apps.

Solution:

  1. Click "More info" in the popup
  2. Click "Run anyway"
  3. The game will launch

❌ "Can't extract ZIP file"

Cause: Corrupted download or missing extraction tool.

Solution:

  • Re-download the ZIP file from itch.io
  • On Windows: Use built-in "Extract All" or download 7-Zip
  • On Mac/Linux: The Archive Utility should work automatically

❌ "Nothing happens when I double-click the executable" (Linux)

Cause: File doesn't have execute permissions.

Solution:

  1. Open Terminal
  2. Navigate to the extracted folder: cd ~/Downloads/Dressmaker
  3. Make executable: chmod +x Dressmaker.x86_64
  4. Run: ./Dressmaker.x86_64

❌ "Game crashes on startup"

Possible causes:

  • Outdated graphics drivers
  • Incompatible OS version
  • Corrupted game files

Solutions to try:

  1. Update your graphics drivers (NVIDIA, AMD, or Intel)
  2. Re-download and extract a fresh copy
  3. Check the itch.io comments for your specific OS
  4. Report the issue on Discord
